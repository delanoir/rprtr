rprtr
===

A freshbooks reporting client built with the [Reactive Cocoa](https://github.com/ReactiveCocoa/ReactiveCocoa) framework and run on iOS7 devices.

Building rprtr
---

```
git clone https://github.com/delanoir/rprtr
cd rprtr
pod install
```
